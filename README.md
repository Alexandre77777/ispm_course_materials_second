## Указания по работе с источниками

> **⚠️ ОБЯЗАТЕЛЬНО:** При написании глав 1-3 необходимо собирать список использованных источников и оформлять ссылки на них в тексте.

**Порядок работы:**
1. При написании **Главы 1** — нумеруйте источники в порядке их упоминания в тексте, и формируйте список в конце главы
2. При написании **Главы 2** — продолжайте нумерацию с последнего номера Главы 1 и дополняйте список
3. При написании **Главы 3** — аналогично продолжайте нумерацию

Таким образом, к моменту выполнения Задания №6 у вас уже будет готовый список литературы, который останется только оформить по ГОСТу.

**Ссылки на источники в тексте оформляются в квадратных скобках:** [1], [2, 3], [4–6]

## Технические требования к оформлению

| Параметр | Значение |
|----------|----------|
| Объём основной части | 60-80 страниц (без приложений) |
| Формат | А4 (210×297 мм) |
| Поля | левое — 30 мм, правое — 15 мм, верхнее и нижнее — 20 мм |
| Шрифт | Times New Roman, 14 пт |
| Междустрочный интервал | полуторный (1,5) |
| Абзацный отступ (красная строка) | 1,25 см |
| Выравнивание текста | по ширине |
| Нумерация страниц | внизу по центру, арабскими цифрами |
| Титульный лист, задание и содержание | не нумеруются (но входят в общий счёт страниц) |
| Рисунки | подписи под рисунком (по центру): «Рисунок 1.1 — Название» |
| Таблицы | подписи над таблицей (по правой стороне): «Таблица 1.1 — Название» |
| Код программы в тексте | моноширинный шрифт (Courier New, 10–12 пт), рамка или заливка фона |
| Ссылки на источники | [1], [2, 3], [4–6] — в квадратных скобках |
